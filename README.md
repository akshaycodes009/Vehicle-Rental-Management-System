# Vehicle-Rental-Management-System (Rentigo)

This repository contains a Vehicle Rental Management System with a React frontend (adapted UI) and a Node.js / Express backend.

Maintainer
 - Akshay Kumar


Run locally

1. Backend

```bash
cd backend
npm install
# create a .env file with MONGO_URL and JWT_SECRET as needed
npm run dev   # runs nodemon index.js
```

2. Frontend

```bash
cd frontend
npm install
npm start
```

Files of interest
- `backend/` — Express API, controllers, models
- `frontend/` — React app
